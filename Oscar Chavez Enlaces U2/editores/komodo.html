Komodo Edit

La idea detrás de Komodo Edit es ofrecer algo poderoso, pero aún así, reteniendo cierto nivel de simplicidad, para que incluso los principiantes puedan entenderlo. Usted puede descargar Komodo Edit para los sistemas operativos de Mac o Windows. Es gratuito y open source, permitiendo que la gente que no necesite las funciones avanzadas, tenga acceso a un editor de texto que funciona de maravilla para terminar proyectos más pequeños.

También, si requiere herramientas más avanzadas como el perfilamiento de código o prueba de unidad, la versión más avanzada de Komodo IDE será más que suficiente. Se ofrece soporte para todos los lenguajes y marcos de trabajo a través del Komodo IDE, haciéndolo ideal para el desarrollo web. También, esta actualización no le costará nada ya que es un proyecto open source.
