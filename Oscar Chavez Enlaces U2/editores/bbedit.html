BBEdit

BBEdit es exclusivamente usado en el sistema operativo de Mac. Este ofrece funciones muy avanzadas, pero también intenta lucir lo más básico posible para ir junto con su acrónimo BB (Bare Bones). BBEdit incluye excelentes funciones como integración Git y autocompletado. El destaque de sintaxis y búsquedas rápidas también proveen, junto con la edición de ventanas que usted pueda dividir y colocar una a lado de la otra para poder editar de una forma más sencilla. La versión actual de BBEdit cuesta $49.99 para recibir una licencia individual. También puede actualizar a nuevas versiones para obtener precios más bajos.

Caracteristicas

BBEdit cuenta con pliegue de código y completado de texto, estos dos hacen que la interfaz luzca más limpia y sea más fácil y rápido program

Funciona con una amplia variedad de lenguajes y tiene revisión de sintaxis para múltiples scripts.

Cuenta con herramientas de búsqueda y reemplazo para ajustar artículos individuales a lo largo de múltiples archivos. También hay varias funciones de navegación al igual que coloreo de sintaxis.

Ofrece herramientas de autocompletado para sugerir cosas como recortes y símbolos.

Puede agregar caracteres especiales a su código e insertar atributos y tags que podrían asistirle mientras escribe su HTML.
